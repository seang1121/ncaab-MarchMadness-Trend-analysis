# Contributing

PRs welcome. Found a bug? Open an issue.

## Getting Started

1. Fork the repo
2. Clone your fork
3. Make your changes
4. Open a PR with a clear description of what you changed and why

## Ideas for Contributions

- Add new prediction models
- Improve historical data coverage
- Better bracket visualization
- Add support for women's tournament data

## Guidelines

- Keep PRs focused — one feature or fix per PR
- Follow the existing code style
- If you're adding a new model, include backtesting results

## Questions?

Open an issue or start a discussion.
